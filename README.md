# index-zhiboba-journal

## 项目简介

`index-zhiboba-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为通用页面集合的存储与展示空间。

仓库中的页面可能涵盖不同主题与用途，主要目的在于提供一种轻量、静态的页面归档方式。

## 目录说明

```
├── index.html          # 入口页面（可选，用于导航或索引）
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 公共资源目录（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要归档目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面共用的样式、脚本及静态资源。

## 页面归档说明

- 每个页面均为完整的独立 HTML 文件，可单独访问。
- 页面之间无强依赖关系，便于分类管理与引用。
- 归档内容以静态形式保存，不依赖后端服务或数据库。
- 页面命名建议采用有意义的英文或拼音，便于识别。

## 维护说明

- 欢迎通过 Pull Request 提交新的页面归档，或对现有页面进行修正。
- 提交前请确保页面内容合规、无恶意代码、无外部追踪脚本。
- 本仓库不承诺对页面内容进行审核或背书，仅作为存储与展示用途。
- 如有大量页面需要归档，建议先通过 Issue 进行讨论。
